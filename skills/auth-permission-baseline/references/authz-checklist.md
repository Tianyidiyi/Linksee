# 鉴权与权限基线检查清单

## 执行步骤

1. 明确身份认证方案和 token 生命周期。
2. 明确授权模型：角色、资源、操作。
3. 每个 API 先写权限断言，再实现业务逻辑。
4. 对输入参数做白名单校验。
5. 敏感操作写审计日志并脱敏。

## 检查清单

- [ ] 未登录请求被拒绝
- [ ] 越权访问被拒绝
- [ ] token 过期与刷新策略清晰
- [ ] 日志无密码、token、隐私数据明文
- [ ] 权限相关用例覆盖正反两种路径

## 常见反例

- 在前端做权限判断，后端不校验
- 用角色名硬编码权限逻辑
- 错误日志输出完整请求体与敏感字段
