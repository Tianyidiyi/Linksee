使用场景：

## 9. 联调结论

结论选项：

- 通过：契约与实现一致，可以进入测试或合并。
- 有条件通过：存在低风险遗留项，已明确负责人和截止时间。
- 不通过：存在阻断问题，需要修复后重新联调。
